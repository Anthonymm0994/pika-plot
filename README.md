## 🛠️ Installation & Quick Start

### Prerequisites
- Rust 1.75+ with WebAssembly target
- GPU with WebGPU support (optional but recommended)
- 8GB+ RAM for large dataset processing

### Installation
```bash
# Clone the repository
git clone https://github.com/user/pika-plot.git
cd pika-plot

# Install dependencies
cargo build --release

# Run the application
cargo run --release
```

### WebAssembly Deployment
```bash
# Install wasm-pack
cargo install wasm-pack

# Build for web
wasm-pack build --target web --out-dir pkg

# Serve locally
python -m http.server 8000
```

### Jupyter Integration
```bash
# Install evcxr Jupyter kernel
cargo install evcxr_jupyter
evcxr_jupyter --install

# Start Jupyter
jupyter notebook
```

## 📚 Quick Examples

### Spatial Analysis
```rust
use pika_engine::spatial_indexing::*;

let mut engine = SpatialIndexingEngine::new();

// Add spatial objects
let objects = load_geospatial_data("cities.geojson")?;
engine.add_objects(objects)?;

// Perform spatial queries
let hotspots = engine.query(&SpatialQuery::HotspotAnalysis { 
    grid_size: 1000.0 
})?;

// Visualize results
let map = SpatialMap::new()
    .add_layer("hotspots", &hotspots)
    .style(MapStyle::professional())
    .interactive(true);

map.show()?;
```

### Graph Analysis
```rust
use pika_engine::graph_analysis::*;

let mut engine = GraphAnalysisEngine::new();

// Load network data
let (nodes, edges) = load_network("social_network.graphml")?;
engine.load_graph(nodes, edges, false)?;

// Analyze network
let centrality = engine.analyze(GraphAnalysisType::PageRank { 
    damping: 0.85, 
    max_iterations: 100 
})?;

let communities = engine.analyze(GraphAnalysisType::LouvainCommunityDetection)?;

// Visualize network
let graph_viz = GraphVisualization::new()
    .layout(ForceDirected::new())
    .color_by_centrality(&centrality)
    .highlight_communities(&communities)
    .interactive(true);

graph_viz.show()?;
```

### Chaos Visualization
```rust
use pika_engine::chaos_visualization::*;

let mut engine = ChaosVisualizationEngine::new();

// Add Lorenz attractor
engine.add_attractor("lorenz", ChaoticAttractor::Lorenz {
    sigma: 10.0, rho: 28.0, beta: 8.0/3.0
});

// Create visualization
let config = ChaosVisualization {
    visualization_type: VisualizationType::Trajectory3D {
        show_points: true,
        show_lines: true,
        trail_length: Some(10000)
    },
    color_scheme: ColorScheme::viridis(),
    resolution: (1920, 1080),
    animation: AnimationConfig::smooth_rotation(),
    ..Default::default()
};

let result = engine.generate_visualization(config)?;
result.save("lorenz_attractor.png")?;
```

### Machine Learning Pipeline
```rust
use pika_engine::advanced_ml::*;

let mut ml_engine = AdvancedMLEngine::new();

// Load and prepare data
let df = DataFrame::from_csv("dataset.csv")?;
let features = FeatureEngineering::new()
    .auto_features(&df)?
    .select_best(50)?;

// AutoML training
let model = ml_engine.auto_ml()
    .target("price")
    .features(&features)
    .train(&df)?;

// Generate insights
let insights = model.explain()?;
let report = AutomatedReport::new()
    .add_model_performance(&model)
    .add_feature_importance(&features)
    .add_insights(&insights)
    .generate()?;

report.export_pdf("ml_analysis_report.pdf")?;
```
